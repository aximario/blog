Container 是一个预设好的环境，它是由 Docker Image 运行而来，每一个 Container 都是 Docker Image 的一个实例。

查看 docker 版本，docker --version

查看 docker 相关信息，docker info

查看 docker image，docker image ls

查看 docker container， docker container ls

Service 是一组生产预设，可以控制 Container 的数量，Service 里的每一个 Container 叫做一个 Task。通过 docker-compose.yml 来配置。

Swarm 是在多台机器上部署多个 Service，部署的机器叫做 node - 节点。
