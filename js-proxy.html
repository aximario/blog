JS 中的 proxy

作者：阿希

更新于：2018-12-17

Proxy 就是代理的意思。基于一个对象创建一个代理对象，这个代理对象可以代理对原对象的部分操作。

语法

                
// target 表示被代理的对象
// handler 表示想要代理的操作，是个对象
const proxy = new Proxy(target, handler)

代理对象能代理的操作一共有 13 种

  1. get
  2. set
  3. has
  4. apply
  5. getPrototypeOf
  6. setPrototypeOf
  7. isExtensible
  8. preventExtensions
  9. getOwnPropertyDescriptor
  10. defineProperty
  11. deleteProperty
  12. ownKeys
  13. construct

get

handler.get 用来代理源对象的属性访问。

                
// 语法
const handler = {
    // target 被代理的对象
    // property 操作的属性
    // receiver Proxy 或者继承 Proxy 的对象
    get(target, property, receiver) { // ... }
}

例如：

                
const handler = {
    get(target, property, receiver) {
        console.log(target, property, receiver);
        return 10;
    }
}
const p = new Proxy({}, handler);
p.a // 10

set

handler.set 用来代理设置源对象的属性。

                
// 语法
const handler = {
    // target 被代理的对象
    // property 操作的属性
    // value 被设置的值
    // receiver 被调用的对象
    get(target, property, value, receiver) { // ... }
}
