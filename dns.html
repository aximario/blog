DNS 简介

作者：阿希

更新于：2019-01-15

DNS 是什么

DNS 全称 Domain Name System，中文译作域名系统。它是互联网的一项服务，用来查找域名对应的 IP。

比如平时我们在浏览器输入 https://www.baidu.com，实际上就是由 DNS 服务找到 baidu.com 对应的 IP 然后再完成的访问。

为什么要用 DNS

访问网络上的主机需要知道主机的地址，也就是 IP 地址，但是 IP 地址是由一串数字组成的，比较难记忆，如果给他们一个名称的话就会好记很多。不仅如此，有了名称之后，即使主机的 IP 地址改变了，也能通过名称找到该主机。

DNS 的实现

要通过域名来找到对应的 IP 就需要通过服务查询，在互联网上有专用的服务器提供 DNS 查询服务，我的计算机会自动的向他们发起 DNS 查询。理论上 DNS 服务器只需要一台就够了，然而事实并非如此。如果全球只用一台 DNS 服务器，虽然在逻辑上很清晰，但是会有下面几个问题：

  • 如果这台服务器出现了故障，整个因特网会随之瘫痪
  • 这台服务器需要处理数亿台设备的 DNS 查询
  • 因为只有一台服务器，对于远处的主机，通信时延会很大
  • 这台服务器要频繁增加，删除，修改域名和 IP 的对应关系

所以实际上的 DNS 服务并非是架设在一台服务器上，而是分层的架设在非常多的服务器上。

根域名服务器：13 个根域名服务器，由 12 个组织管理，分布于全球各地，截止 2019 年 1 月，现有 900 多台根域名服务器（包括镜像服务器）。

http://www.root-servers.org

  • 顶级域名服务器：com, net, org, name 等，也包括一些国家域，cn，cc 等
  • 权威域名服务器：一些大学或者大公司维护的域名服务器
  • 本地域名服务器：机构或者居民 ISP 提供的域名服务器
